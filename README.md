# App de Estudio de Tributacion Local

Aplicacion web sencilla para practicar preguntas tipo test sobre tributacion local.

## Contenido

La app trabaja con preguntas basadas en:

- Texto Refundido de la Ley Reguladora de las Haciendas Locales
- Materiales de sistema tributario local

## Funcionalidades

- seleccion de bloque
- modo secuencial
- modo aleatorio
- simulacro de 20 preguntas
- correccion inmediata
- marcador de aciertos y errores

## Ejecutar

```bash
python -m pip install -r requirements.txt
python -m streamlit run streamlit_app.py
